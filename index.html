Projeto YANA

Projeto desenvolvido em grupo para a matéria Análise e Projeto de Sistemas, do curso Análise e Desenvolvimento de Sistemas da UNICID. O objetivo do Y.A.N.A é democratizar o acesso ao atendimento psicológico, utilizando para isso de uma plataforma online que garante a anonimidade dos usuários e permite que todos possam conversar com profissionais da área sobre seus problemas. O site foi utiliza das linguagens HTML, CSS, JavaScript e PHP.
